# Secure File Transfer Monitoring System

## Project Description
A Flask-based web application that allows users to upload, monitor, and manage files securely with safety classification.

## Features
- User Login System
- File Upload & Download
- Safe / Unsafe file detection
- File management dashboard
- Activity tracking UI

## Technologies Used
- Python (Flask)
- HTML, CSS
- Jinja2 templates

## Deployment
Hosted on PythonAnywhere
